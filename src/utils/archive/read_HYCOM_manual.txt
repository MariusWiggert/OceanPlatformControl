HYCOM creates the NetCDF files from which we access data.

Just select the file wanted from the HYCOM server and then either
a) Downloaded as .nc or .nc4 (newer version) files
via FTP or more intuitive for selecting what exactly the NetcdfSubset website.
=> read in via
from netCDF4 import Dataset
data = Dataset("data/2019.nc4", "r", format="NETCDF4")

b) Load the file via URL & OpenDAP
select the subset & copy the data link
=> read in via
from utils.opendap import opendap
url_grid= r'http://tds.hycom.org/thredds/dodsC/GOMu0.04/expt_90.1m000/data/hindcasts/2019?depth[0:1:39],lat[0:1:345],lon[0:1:540],time[0:1:8255],water_u[0:1:1][0:1:1][0:1:345][0:1:540]'

# Get grid data
grid = opendap(url_grid)
G_lat = grid.variables['lat']
G_lon = grid.variables['lon']
G_water_u = grid.variables['water_u']
